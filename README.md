My Travel Journal is a simple and responsive **React** application that displays travel destinations in a clean, card-based layout.  
Each entry includes an image, country, location name, travel dates, description, and a direct link to view the location on Google Maps.

This project focuses on **React fundamentals**, component reuse, and clean UI styling using **HTML and CSS**.

## âš™ï¸ How the App Works

- Travel data is stored in a `data.js` file as an array of objects
- The `App` component maps over the data and renders a `Card` component for each destination
- Props are used to pass destination details to each card
- The header is a separate reusable component
- Styling is handled entirely with CSS for layout and spacing
